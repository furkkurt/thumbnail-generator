# Blog Thumbnail Generator

Monorepo: `frontend` (Astro, Node adapter, Tailwind, Actions) ve `backend` (.NET 10 API) Türkçe blog metnini OpenRouter (varsayılan: **Gemini** slug’ı) ile tek bir **İngilizce** görsel promptuna çevirir; bu prompt konsolda (`OpenRouter İngilizce görsel prompt: …`) loglanır, ardından VCTRTN vektör soneki eklenir ve Replicate ile görsel üretilir. **16:9** için `Replicate:SendAspectRatio` genelde **true** ve `AspectRatio` **16:9** olmalı; kullandığınız Replicate modelinin API şemasında `aspect_ratio` alanı yoksa **422** alırsınız — o zaman model dokümantasyonuna göre giriş anahtarını/değerini güncelleyin ([`ReplicatePredictionService`](backend/src/ThumbnailGenerator.Api/Services/ReplicatePredictionService.cs)). Ortamla geçersiz kılma: `REPLICATE_SEND_ASPECT_RATIO`.

Kök dizinde `.env` şablonu için [.env.example](.env.example) dosyasına bakın; `REPLICATE_API_TOKEN`, `REPLICATE_FLUX_MODEL_VERSION` ve **`OPENROUTER_API_KEY`** zorunludur (görsel prompt üretimi için). İsteğe bağlı: `OPENROUTER_MODEL` (boşsa `appsettings.json` içindeki `OpenRouter:Model`). OpenRouter arayüzünden kopyalanan model kimliğinde başta **`~`** varsa (ör. `~google/gemini-pro-latest`) API’ye de aynen öyle verin; tildesiz `google/gemini-pro-latest` bazen geçersiz model (400) döner — backend eski değeri otomatik `~` ile düzeltir. Token ve sürüm değerlerinde **tırnak kullanmayın** (veya kod dış tırnakları temizler). Yerelde API’yi `backend/src/ThumbnailGenerator.Api` içinden `dotnet run` (port 5080), arayüzü `frontend` içinden `npm run dev` (port 4321) ile çalıştırın; ikisini birlikte için kökte [`run.sh`](run.sh) kullanabilirsiniz (API arka planda başlar). Token doğrulaması: `curl https://api.replicate.com/v1/account -H "Authorization: Bearer $REPLICATE_API_TOKEN"`.

**401 / “valid authentication token”:** API konsolunda `Replicate token yüklendi` veya `REPLICATE_API_TOKEN boş` loguna bakın. Development’ta `GET http://localhost:5080/api/diagnostics/replicate` ile `tokenLooksLikeReplicate` ve uzunluk kontrol edin. `.env` içinde **tek satır** kullanın: `REPLICATE_API_TOKEN=r8_...` — satıra `export` kelimesini veya kabuk komutunu **değer olarak** yapıştırmayın (logda önek `export` görünüyorsa format hatalıdır). Replicate hesabında yeni token: [API tokens](https://replicate.com/account/api-tokens). Kabukta eski export için: `unset REPLICATE_API_TOKEN`.

**Terminal “exit code 130”:** Genelde **Ctrl+C** ile süreci kestiğiniz anlamına gelir; HTTP 502 bunu tetiklemez. İki servisi birlikte için kökte `./run.sh` kullanın (`cd "$ROOT/frontend"` satırını başka terminale kopyalamayın; `ROOT` yalnızca script içindedir).
